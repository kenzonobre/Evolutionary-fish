# Evolutionary-fish
Shoals of fish based on boids algorithm that evolve through a genetic algorithm

<p align = "center">
  <img width = "500" src = "https://github.com/kenzonobre/Evolutionary-fish/blob/main/assets/simulation.gif">
</p>

The goal of this project was to evolve species of fish in an hostile environment consisting of sharks. In order accomplish this evolution, it was developed an genetic algorithm that prioritizes the number of live fish.
